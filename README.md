## ✨ About the 3D Portfolio

This website is designed as an immersive interactive interface using **Three.js (React Three Fiber)** and **Framer Motion** to create a dynamic space-station atmosphere that changes color schemes as you navigate through different sections.

### Interactive Features:
* **Interactive 3D Particles:** Moves and tilts dynamically with your cursor for a parallax effect.
* **Responsive Sections:** Swipe or scroll to move between Hero, About, Skills, Experience, Projects, and Contact sections.
* **Fluid Transitions:** Custom cursor trail and smooth page animations.
